# MindPlay Arena 🎮

MindPlay Arena is an interactive mini-game platform built using React, TypeScript, and Vite.

## 🚀 Features

- Word Scramble
- Riddles Challenge
- Guess the Word
- Rapid Fire Quiz
- Responsive Design
- Smooth Animations

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

```bash
npm install
npm run dev
